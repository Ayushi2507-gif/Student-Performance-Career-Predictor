##  2. Project Description
Traditional educational tracking systems often keep marks and attendance scattered, making it hard to evaluate a student's true potential. Moreover, rigid systems penalize students for low attendance without considering personal struggles.

To solve this, this project introduces a **Dual-Mode System**:
1. **12th Pass Career Counseling:** Evaluates core stream subjects along with communication skills to guide high school graduates toward tailored higher-education options (PCM, PCB, Commerce, Arts).
2. **University Analytics Dashboard:** Integrates attendance monitoring and subject performance into centralized, colorful visual charts for faculty and students.
3. **Resilience Logic:** A unique feature that analyzes the contextual reasons behind low attendance (such as family or medical emergencies) to reward a student's perseverance rather than delivering unfair scolds.

---

##  3. Technologies Used
The application is built using a modern Python-based data science stack:
- **Python:** For drafting the core tiered-rules and contextual analysis engine.
- **Streamlit:** For designing the interactive web application layout and side navigation menu.
- **Pandas:** For structured data retrieval and student record management.
- **Matplotlib:** For rendering real-time performance analytics and bar charts.
